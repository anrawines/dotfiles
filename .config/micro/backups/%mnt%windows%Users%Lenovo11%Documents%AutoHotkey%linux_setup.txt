MINIMAL INSTALL LINUX
tty ctrl + alt + f1 .. f6

Tahap 1: Fondasi Sistem (Core)
	install video driver ~ mesa 
	install network manager ~ networkmanager
	install fonts ~ ttf-jetbrains-mono-nerd , noto-fonts
	
Tahap 2: Lapisan Grafis & Input (Display)
	install display server ~ x org or wayland
	install display manager ~ lightdm 
	install window manager ~ i3 / sway

Tahap 3: Service Pendukung (Background Daemons)
	install polkit agent ~ lxsession / polkit-gnome
	install notification daemon ~ dunst / mako 
	install compositor ~ picom
	install audiomanager ~ pulseaudio / pipewire & pipewire-pulse
	install usb mounter ~ udiskie
	install shortcut manager ~ sxhkd

Tahap 4: Personalisasi & App (Frontend)
	Appearance
		gtk manager : lxappearance
		qt manager : qt5ct , qt6ct
	Bar & Wallpaper
		topbar ~ polybar, waybar
		img backround  : feh
		colors themes : pywall16, matugen
